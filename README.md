
# CryptedVault, a local encrypted data storage solution.

### This project is made as an example of Versapy project and for personnal use, if you intent to store sensible data make sure that the encrypting and security rate feats your standards.

## About the App

#### This project uses Versapy, a python-desktop app framework. See more on the [official](https://github.com/Soli-64/VersaPy/) repo.

### Frontend

#### A basic Vite+React+taiwind UI, using shadcn/ui for modern components.
#### Webview handled by Versapy. (via PyWebView)

### Backend

#### API endpoints and front connection is handled by Versapy (via Sockets).

### Data Encryption

#### Every data stored by the app in a vault is encrypted with the user password thanks to cryptography and argon2 python modules. Make sure to not lose it, or the data we'll be lost too.
